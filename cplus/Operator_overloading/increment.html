重载递增运算符（++）¶

Table of Contents

  • 重载递增运算符（++）

    • 概述

    • 示例

概述¶

  • 前置递增（++i） ：返回的是引用。

  • 后置递增（i++） ：返回的是对象（值）。

示例¶

class MyInteger {
    int m_Num;
public:
    friend std::ostream& operator<<(std::ostream& cout, MyInteger myInt);

    MyInteger() {
        this->m_Num = 0;
    }

    // 前置++
    MyInteger& operator++() {
        this->m_Num++;
        return *this;
    }

    // 后置++
    MyInteger operator++(int) {
        MyInteger tmp = *this;
        this->m_Num++;
        return tmp;
    }
};

std::ostream& operator<<(std::ostream& cout, MyInteger myInt)
{
    cout << myInt.m_Num;
    return cout;
}
