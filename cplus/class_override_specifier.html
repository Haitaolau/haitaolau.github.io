概述¶

Specifies that a virtual function overrides another virtual function. override 关键字的作用是检查派生类中函数是否在重写基类虚函数，如果不是重写基类虚函数，编译器就会报错。

示例¶

class Base
{
public:
    virtual void func();
    void show();
};
class Derive:public Base
{
public:
    void func() override;   // 可通过编译
    void show() override;   // 不能通过编译：show() 不是虚函数
};

输出：

class_overiding.cpp:15:7: error: 'void Derive::show()' marked 'override', but does not override
   15 |  void show() override;
      |       ^~~~
