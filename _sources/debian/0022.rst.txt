===========================================
To remove the stale remote tracking branch
===========================================

``git branch -D hliu4/dev`` only deletes the local branch, not the remote tracking reference. You still have the remote tracking branch origin/hliu4/dev in your local 
repository.

To remove the stale remote tracking branch, use:

.. code-block:: bash 

   git remote prune origin


This will remove all remote tracking branches that no longer exist on the remote repository.

Alternatively, you can remove the specific remote tracking branch:

.. code-block:: bash 

   git branch -dr origin/hliu4/dev


After running either command, git branch -r should no longer show origin/hliu4/dev.

The git remote prune origin approach is generally preferred as it cleans up all stale remote tracking branches at once.

The result:

.. code-block:: bash 

   wrsadmin@pek-hliu4-d2:~/workspace/dist2$ git branch -r
   origin/HEAD -> origin/main
   origin/aria
   origin/hliu4/aria
   origin/hliu4/dev
   origin/main
   origin/wentao-dev
   upstream/aria
   upstream/main
   upstream/wentao-dev
   wrsadmin@pek-hliu4-d2:~/workspace/dist2$ git remote prune origin
   Pruning origin
   URL: https://gitlab.com/haitailau/dist2.git
   * [pruned] origin/hliu4/dev
   wrsadmin@pek-hliu4-d2:~/workspace/dist2$ git branch -r
   origin/HEAD -> origin/main
   origin/aria
   origin/hliu4/aria
   origin/main
   origin/wentao-dev
   upstream/aria
   upstream/main
   upstream/wentao-dev
   wrsadmin@pek-hliu4-d2:~/workspace/dist2$
